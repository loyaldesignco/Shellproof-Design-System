# Shellproof Style Guide

This style guide is token-driven and references all variables using the `--sp-` prefix.

## Typography
- Headings use `--sp-font-heading`
- Body uses `--sp-font-body`
- Mono uses `--sp-font-mono`

## Buttons
Primary buttons use:
- `--sp-core-accent-color_accent-primary`
- `--sp-radius-md-radius`

Secondary buttons:
- Transparent background
- `--sp-border-color_border`

## Forms
Inputs use:
- `--sp-card_card-primary-bg`
- `--sp-border-color_border`
- `--sp-radius-md-radius`

## Colors
All color values are sourced from `tokens/css-variables.css`.
Never hardcode colors. Always reference variables.
