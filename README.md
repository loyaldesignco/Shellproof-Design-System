# Shellproof AI Brand System

Shellproof Security's AI brand infrastructure repository. This system gives AI tools, designers, and developers a shared source of truth for content, layout, styling, and component decisions.

## What this repo contains
- Brand profile and voice rules
- Color, typography, and design tokens
- Layout patterns and component library
- AI system instructions and model-specific guides
- Prompt library for defense and CMMC-focused content
- Webflow, CSS, Tailwind, and Figma token files
- Document templates and example outputs
- Asset structure for logos, diagrams, photography, social graphics, and fonts

## How to use
1. Load `AI_BOOT_SEQUENCE.md`
2. Load the system files listed in `AI_SYSTEM_PROMPT.md`
3. Choose a prompt from `AI_PROMPTS/` or use `AI_CONTENT_ENGINE.md`
4. Generate output using the correct structure from `LAYOUT_PATTERNS.json`

## Asset placement
- Logos: `ASSETS/logos/`
- Icons: `ASSETS/icons/`
- Diagrams: `ASSETS/diagrams/`
- Photography: `ASSETS/photography/`
- Social graphics: `ASSETS/social/`
- Font files: `ASSETS/fonts/`
