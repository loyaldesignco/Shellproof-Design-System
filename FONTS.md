# Font Usage

## Heading Font
Obviously Wide Medium

Used for:
- H1
- H2
- Hero headlines
- Section titles

## Body Font
Roboto Mono

Used for:
- body text
- UI labels
- technical content
- documentation

## File Placement
Place licensed webfont files in:
- ASSETS/fonts/obviously-wide/
- ASSETS/fonts/roboto-mono/

Recommended formats:
- .woff2
- .woff
- .otf
